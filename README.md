# 🐄 Cattle Breed Identification System

The **Cattle Breed Identification System** is a web-based platform designed to help **farmers identify cattle breeds using images** and manage their cattle records digitally.  
The system uses a **deep learning model (ResNet50)** for breed identification and provides features like **cattle registration, unique ID assignment, authentication, and cattle management**.

---

## 🚀 Features

- 📷 **Cattle Breed Identification**
  - Farmers can identify the breed of their cattle by **uploading or clicking an image**
  - Uses a **ResNet50-based deep learning model** trained on cattle breed images

- 🆔 **Cattle Registration**
  - Farmers can register their cattle in the system
  - Each registered cattle is assigned a **unique 12-digit Cattle ID**

- 📋 **Cattle Management**
  - View all registered cattle
  - Manage and track cattle details from a single dashboard

- 🔐 **Authentication System**
  - Secure user authentication
  - Farmers can log in and manage only their own cattle data

- 🧠 **AI-Powered Backend**
  - Breed prediction is handled by a trained **ResNet50 model**
  - Integrated with the web application using REST APIs

---

## 🛠️ Technology Stack

- **Deep Learning**: ResNet50
- **Backend**: Python (Flask/FastAPI)
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQL-based database
